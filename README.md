# 셀레니움을 이용한 프로그래머스 채용정보 크롤러

## ✅ 크롤러 사용 방법
1. github clone
2. `build.gradle` 실행
3. parallel.main 실행
4. 희망 직무 선택 [입력]
5. 희망 경력 선택 [입력]
6. 희망 연봉 선택 [입력]
7. 모니터 해상도 입력 [입력]
8. 크롤링한 데이터를 저장 할 파일 명 이름 입력 (확장자 제외) [입력]
9. 기다림....
10. `오늘날짜/입력한 파일 명` 엑셀 파일 생성, 예시 `2023-11-26/result.xlsx`


## ❗️ 주의
- 크롤링 도중에 `mock` 디렉토리가 생기는데, 이 디렉토리를 도중에 열어놓으면 크롤링 결과가 저장되지 않는다.
